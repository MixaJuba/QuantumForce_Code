- OBD-II — On-Board Diagnostics II

- DoIP — Diagnostics over IP (ISO 13400)
- J2534 — Pass-Thru API
- DTC — Diagnostic Trouble Code
- ECU — Electronic Control Unit
- VIN — Vehicle Identification Number
- ASIL — Automotive Safety Integrity Level
