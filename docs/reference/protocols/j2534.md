# SAE J2534 (Pass-Thru)

## Огляд
- Стандарт API для підключення до OEM адаптерів
- Підтримка множинних протоколів: ISO9141, KWP2000, CAN, J1850

## Використання
- Вибір каналу, налаштування baudrate, фільтри
- Thread-safe виклики, контроль таймінгів
