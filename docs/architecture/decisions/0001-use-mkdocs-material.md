## Context
Need fast, modern docs with AI-friendly Markdown and Mermaid.

## Decision
Adopt MkDocs Material with mermaid2, minify, git-revision-date plugins.

## Consequences
- Quick authoring and CI integration
- Easy PR previews and GitHub Pages
