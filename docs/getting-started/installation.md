# Встановлення

## Локально
```bash
pip install mkdocs-material mkdocs-mermaid2-plugin mkdocs-minify-plugin mkdocs-git-revision-date-localized-plugin
```

## Попередній перегляд
```bash
mkdocs serve
# http://localhost:8000
```
